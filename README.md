# Iris Flower Classification 🌸

## Project Overview
This project uses Machine Learning to classify iris flowers into different species based on flower measurements. The classification is performed using supervised learning algorithms.

The model predicts the species of iris flowers among:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

---

## Dataset Information
The Iris dataset contains:
- 150 rows
- 4 input features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 output classes
